 ## Start
 ### Server

Create the new local DataBase:
 ```sh
$ npm install
$ npm run db:migrate
```
Start server:
```sh
$ npm run dev
```

### Web 
Start a local web client:
```sh
$ npm install
$ npm run dev
```
