## Description 
A platform that allows you to post ads to find a duo to play with.

## Stack/Tools
*Back End:*
 - Node.js    
 - Prisma

*Front End:*
 - React
 - ViteJs
 - Radix UI
 - Tailwind 

 ## Start
 ### Server

Create the new local DataBase:
 ```sh
$ npm install
$ npm run db:migrate
```
Start server:
```sh
$ npm run dev
```

### Web 
Start a local web client:
```sh
$ npm install
$ npm run dev
```
